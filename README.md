# Staf_Cadastrar_Pessoa (Prática 8)

### Clone o repositório utilizando git:

- git clone https://github.com/anaclaraosc/Staf_Cadastrar_Pessoa.git

- cd Staf_Cadastrar_Pessoa

### Instale as dependencias:

- npm install

### Startando a aplicação:

- npm start

<http://localhost:8080/>.
